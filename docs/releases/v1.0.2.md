# SafeVault v1.0.2

SafeVault v1.0.2 is a small safety and wording correction for protected-folder
management. Storage, snapshot, restore, and Windows startup behavior are
unchanged from 1.0.1.

## What Changed

- Recovery Home now has a safe “Stop automatic protection” action. It disables
  monitoring while preserving snapshots, versions, and recovery points.
- The existing destructive action is now labeled “Remove history permanently.”
  It clearly explains that database recovery indexes are deleted and therefore
  no longer restorable through SafeVault.
- Destructive removal still requires a preview and typed root ID or full path.

## Upgrade Notes

Install `SafeVaultSetup.exe` over 1.0.0 or 1.0.1. Existing vault data and backup
configuration remain in place.
