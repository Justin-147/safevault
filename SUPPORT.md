# SafeVault Support

- Installation and usage questions: use [GitHub Discussions](https://github.com/Justin-147/safevault/discussions).
- Reproducible product defects: open a bug report using the issue template.
- Feature proposals: open a feature request or start a Discussion first.
- Security vulnerabilities: follow [SECURITY.md](SECURITY.md) and do not open a
  public issue.

Before asking for help, check the [FAQ](docs/FAQ_EN.md),
[中文常见问题](docs/FAQ_ZH.md), and
[Troubleshooting guide](docs/zh/TROUBLESHOOTING.md).
